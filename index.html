HTML

  • Hypertext Markup Language (HTML) gives content stucture.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title for the page.
  • The link element connects external resources to the html document.
  • The meta data element defines metadata such as character set, description, keywords, author, and viewport
  • The scrip element embeds client-side scripts in html document.

CSS

  • A margin indicates how much sapce we want arount the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Three ways to style a webpage with CSS:
    • Inline CSS
    • Internal CSS Style Sheet
    • External CSS Style Sheet
